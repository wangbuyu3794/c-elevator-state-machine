# C Elevator State Machine

## 项目简介

这是一个使用 C 语言实现的单电梯状态机模拟项目。

项目以“电梯运行控制”为场景，重点练习 C 语言中的结构体、枚举、函数拆分、头文件与源文件分离、状态机、事件处理、请求调度和异常安全机制。

本项目不是为了模拟真实商用电梯的全部复杂逻辑，而是通过一个接近现实的控制系统案例，训练如何把现实流程抽象成 C 语言程序。

## 项目目标

本项目计划实现一个简化但结构清晰的电梯模拟系统。

核心目标包括：

- 支持单台电梯运行；
- 支持楼层范围：地下 1 层到地上 34 层；
- 使用 -1 表示地下 1 层；
- 不存在 0 楼；
- 支持多个楼层请求；
- 采用“顺路优先，无顺路请求时最近优先”的调度策略；
- 模拟电梯上行、下行、开门、关门、停留等流程；
- 后续加入超重机制；
- 后续加入门阻挡检测与重新开门机制；
- 后续加入故障反应机制；
- 后续加入停电与恢复供电后的安全恢复机制；
- 后续加入管理员紧急暂停机制；
- 输出清晰的运行日志，便于观察状态变化。

## V1 功能

当前版本实现了一个最小可运行的单电梯状态机模拟程序。

已支持：

- 楼层范围：`-1` 到 `34`；
- `-1` 表示地下 1 层；
- 不存在 `0` 楼；
- 单台电梯；
- 多楼层请求；
- 请求去重；
- 顺路优先，无顺路请求时最近优先；
- 基础开门、停留、关门流程；
- 固定时间模拟；
- 命令行菜单交互；
- 清晰运行日志。

V1 暂不实现：

- 超重；
- 门阻挡；
- 故障；
- 停电恢复；
- 管理员暂停；
- 多电梯群控。

## 编译方式

在项目根目录执行：

```powershell
gcc -Iinclude src/main.c src/elevator.c -o elevator.exe
```

## 运行方式

```powershell
.\elevator.exe
```

## 菜单说明

程序启动后可以选择：

- `1` 添加楼层请求；
- `2` 运行一步；
- `3` 一直运行到所有请求完成；
- `4` 查看当前电梯状态；
- `0` 退出程序。
